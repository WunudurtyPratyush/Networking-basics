# Networking-basics for an interview
What is networking : A Coputer network is a collection of interconnected devices that share resources and information. These devices can include computers, servers, printers and other hardware devices.
Network: A group of connected computers and devices that can communicate and share data with eachother.
Node: Any device that can send, receive or forward data in a network. This includes laptop, mobiles, printers, servers etc.
Networking Devices: Devices that manage and support networking functions. These include routers, switches, hubs and access points.
Transmission media : The path through which data travels from one device to another.
Types of transmission media: 
A) Guided Wired Media :  Data signals travel through a physical path.
Twisted pair cable: two copper wires twisted to reduce interference (Used in LAN, telephones, Ethernet cables
Coaxial cables : Single copper wire + Insulation + Metal shield  (used in Cable Tv, CCTV, older internet connection.
Optical Fiber cable: Uses light instead of electrical signals. (Very high speed, long-distance internet, data centres)

B) Unguided media: Data travels through air (no wires) using radio or light waves.
